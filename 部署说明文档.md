# 课堂电子宠物系统 — 部署说明文档

> 本文档帮助你将系统完整部署到 [ct8.pl](https://www.ct8.pl) 免费主机，完成后任何人通过浏览器即可使用。

---

## 一、准备工作

### 1.1 注册 ct8.pl 账号

1. 打开 [https://www.ct8.pl](https://www.ct8.pl)，点击右上角 **Register** 注册账号
2. 填写用户名、邮箱、密码，完成注册
3. 登录后进入控制面板 [https://panel.ct8.pl](https://panel.ct8.pl)

---

## 二、创建数据库

1. 在控制面板左侧菜单找到 **MySQL Databases**，点击进入
2. 在 **Create new database** 区域：
   - 填写数据库名称，例如 `pet_system`（实际名称会带上你的用户名前缀，如 `m12345_pet_system`）
   - 点击 **Create** 创建
3. 在 **Add new database user** 区域：
   - 填写用户名，例如 `petadmin`
   - 填写密码（请记下来，后面要用）
   - 点击 **Create**
4. 在 **Add user to database** 区域：
   - 选择刚才创建的用户和数据库
   - 权限选 **All privileges**
   - 点击 **Add**

完成后你会得到以下信息（请记录）：

| 项目 | 示例值 |
|------|--------|
| 数据库主机 | `mysql.ct8.pl` |
| 数据库名称 | `m12345_pet_system` |
| 数据库用户名 | `m12345_petadmin` |
| 数据库密码 | 你设置的密码 |

---

## 三、修改数据库配置文件

打开项目文件夹里的 `api/config.php`，将以下几行改成你自己的数据库信息：

```php
define('DB_HOST', 'mysql.ct8.pl');          // 不需要改
define('DB_PORT', 3306);                     // 不需要改
define('DB_NAME', '这里填你的数据库名称');    // 例如：m12345_pet_system
define('DB_USER', '这里填你的数据库用户名'); // 例如：m12345_petadmin
define('DB_PASS', '这里填你的数据库密码');   // 你设置的密码
```

保存文件。

---

## 四、上传网站文件

### 4.1 进入文件管理器

1. 在 ct8.pl 控制面板左侧找到 **File Manager**，点击进入
2. 进入 `public_html` 文件夹（这是网站根目录）

### 4.2 上传文件

将整个 `pet-system` 文件夹内的所有内容上传到 `public_html` 目录下。

上传后目录结构应该是这样：

```
public_html/
  ├── index.html
  ├── api/
  │   ├── config.php
  │   ├── init.php
  │   ├── auth.php
  │   ├── tasks.php
  │   ├── students.php
  │   └── admin.php
  ├── js/
  ├── styles/
  └── screenshots/
```

> **提示**：可以先把 `pet-system` 文件夹打包成 `.zip`，用文件管理器的上传功能上传后再解压，会更快。

---

## 五、初始化数据库

文件上传完成后，在浏览器地址栏输入：

```
https://你的用户名.ct8.pl/api/init.php
```

例如：`https://m12345.ct8.pl/api/init.php`

如果看到如下返回内容，说明数据库初始化成功：

```json
{"success":true,"msg":"✅ 数据库初始化完成！所有表已创建，初始数据已写入。请访问完成后删除或重命名 init.php 文件以防止重复调用。"}
```

> ⚠️ **重要**：初始化成功后，请立即在文件管理器中**删除或重命名** `api/init.php` 文件，防止他人重复调用重置数据！

## 八、常见问题

**Q：上传后打开网页空白或报错？**
- 检查 `api/config.php` 里的数据库信息是否填写正确
- 确认数据库初始化（访问 `init.php`）是否成功

**Q：init.php 返回"数据库连接失败"？**
- 检查数据库主机是否为 `mysql.ct8.pl`
- 检查数据库名、用户名、密码是否和面板里的完全一致（注意有用户名前缀）

**Q：学生/教师登录后数据不显示？**
- 按 **Ctrl+F5** 强制刷新页面清除浏览器缓存
- 检查 `init.php` 是否成功执行

**Q：ct8.pl 免费主机有什么限制？**
- 免费版有每月流量限制，适合班级内部小规模使用
- 如需长期稳定使用，可考虑升级付费套餐
